HTML

HTML is the main markup language for describing the structure of Web pages.

HTML является основным языком разметки для описания структуры веб-страниц.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document.

Cascading Style Sheets (CSS ) это язык стилей используется для описания внешнего вида и форматирования документа.

Bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Bootstrap cамый популярный фреймворк для разработки адаптивных и мобильных web-проектов.

JS

JavaScript is most commonly used as a client side scripting language, is a lightweight, interpreted, object-oriented language with first-class functions.

JavaScript часто используеться в качестве клиентской стороны, скриптовый язык, интерпретированый, объектно-ориентированный язык.

D3.js

D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS.

D3.js библиотека JavaScript для манипулирования. D3 помогает перенести данные для использования в HTML , SVG и CSS.

GitHub

GitHub is a Web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality of Git.

GitHub это веб-хостинг Git репозиторий , который предлагает все распределены контроля версий и управления источника кода ( SCM), функциональность Git.
